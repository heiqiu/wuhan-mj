## 💡 常用技巧

### 技巧1: 优先级排序
```
优先打出:
1️⃣ 孤张字牌 (如单张东南西北)
2️⃣ 孤张边张 (如单张1万、9筒)
3️⃣ 孤张中张 (如单张5条)

优先保留:
1️⃣ 刻子 (三张相同)
2️⃣ 对子 (两张相同)
3️⃣ 搭子 (相邻的牌)
```

### 技巧2: 看反馈学习
```
每次打完牌后:
✅ 查看"你的选择"排名
✅ 阅读"改进建议"
✅ 对比"所有选项排名"
✅ 理解为什么这样打
```

### 技巧3: 利用历史
```
定期查看"📊 查看历史":
✅ 看平均得分趋势
✅ 找出常犯错误
✅ 检查是否进步
```

## ⚠️ 常见问题

### Q: 为什么系统说我选错了?
```
A: 查看"原因"说明,通常是:
- 打出了有搭子潜力的牌
- 保留了孤张边张
- 忽视了对子价值
```

### Q: 对子一定要保留吗?
```
A: 不一定!
- 字牌对子 → 优先保留
- 数牌对子 → 看有无搭子
- 已有多个对子 → 可以拆一个
```

### Q: 边张都要打吗?
```
A: 不一定!
- 孤立边张 → 优先打出
- 有搭子的边张 → 可以保留
- 例如: 1万2万 比 单张1万 价值高
```
